Speech Error Analysis

Data Analysis

This section allows you to input a CSV file containing various results of a speech recognition experiment. Based on this file, it generates interactive graphs, which can be sliced-diced and compared.

Out of Vocabulary

This section allows you to input three text files for (1) training data, (2) testing data and (3) dictionary. Based on these files, it generates a list of words which are out of vocabulary.

Substitution Words

This section allows you to input detailed output of sclite tool (dtl file). Based on this file, it generates a list of most substituted words.
